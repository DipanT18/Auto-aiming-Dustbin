# Build timeline

Suggested milestones for a semester-style project. Adjust dates to your course schedule.

| Phase | Deliverable |
|-------|-------------|
| Week 1–2 | Repo, hardware list, basic camera capture |
| Week 3–4 | Motor control firmware; manual serial tests |
| Week 5–6 | Object detection baseline; tracking in frame |
| Week 7–8 | Serial bridge from vision to motors; open-loop aiming |
| Week 9–10 | Calibration; trajectory prediction integration |
| Week 11–12 | Tuning, demo video, documentation freeze |

## Risks

- Shipping delays on drivers/camera
- USB bandwidth on Pi with high resolution at high FPS
- Outdoor lighting if demo is not indoors

## Demo checklist

- [ ] Repeatable throw zone marked on floor
- [ ] Backup recorded video if live demo fails
- [ ] Labeled emergency stop / power switch
